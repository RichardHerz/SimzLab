Dimensional Analysis

Dimensional analysis involves the use of units to determine if a problem has been solved correctly.  Units are extremely important when it comes to problem solving and understanding what a problem means.  For example, you may be told that someone you know was able to run a mile in five.  This would be an impressive speed if it were five minutes, but if it were five hours, then it is very slow.  The statement means nothing without units. 

The idea of units becomes more complicated because there are Metric units as well as English units.  Most of the world uses the metric system; however, here in America, we use primarily English but in some cases both.  The English system is more familiar to us.  We have some idea what a pound weighs but less of an idea about a kilogram.   Similarly, we know about how far a mile is but not a kilometer, and how much a gallon is but not a liter.  It is important to understand both systems because you want to be able to share knowledge with others around the world.  

Sometimes it is necessary to convert from one system of units to another.  Often this happens when you are collecting data from different sources.  In addition, there is more than one unit in each system for measuring the same property.  For example, pressure can be measured in atmospheres, Pascal’s, bars, mmHg, psi, torr, and several others.  It is also important to be able to convert between each form.  Unit conversion tables can be found in most science textbooks.  An abbreviated list is given below.   

Mass

1 kg = 1000 g = 2.20462 lb = 35.27392 oz.

1 lb = 16 oz = 453.593 g

Volume

1 m3 = 1000 L = 106 cm3 = 106 mL = 35.3145 ft3 = 264.17 gal = 1056.68 qt

1 ft3 = 1728 in3 = 7.4805 gal = 0.028317 m3 = 28,317 cm3 = 28.317 L

Length

1 m = 100 cm = 1000 mm = 39.37in = 3.2808 ft = 1.0936 yd = 0.0006214 mile

1 ft = 12 in = 0.333 yd = 0.3048 m = 30.48 cm

Force

1 N = 1 kg-m/s2 = 105 g-cm/s2 = 0.22481 lbf

1 lbf = 32.174 lbm-ft/s2 = 4.4482 N

Pressure

1 atm = 1.01325*105 N/m2 (Pa) = 101.325 kPa = 1.01325 bar = 760 mmHg = 14.696 lbf/in2 (psi)

Energy

1 J = 1 N-m = 2.778*10-7 kW-h = 0.239001 cal = 0.7376 ft-lbf = 9.486*10-4 Btu

Power

1W = 1 J/s = 0.23901 cal/s = 0.7376 ft-lbf/s = 9.486*10-4 Btu/s = 1.341*10-3 hp

 

When you are solving a problem using unit analysis, it is important that you know what your final units should be.  For example, if you want to find the velocity a car is traveling, you know you want your units to be in terms of distance per time.  If you were to get an answer in terms of distance per time squared, you would know you found the acceleration, not the velocity.  It is important to keep units neat and make sure you write clearly what is in the numerator and what is in the denominator.  If done properly, unit analysis can be an easy way to check your work.

 

Examples

  1. A train traveling from Chicago to Milwaukee is going 65 miles/hour.  A train traveling from Paris to Berlin is going 90 km/hour.  Which is going faster? 

Solution: To answer this problem, you must have the two speeds measured in the same units so that you can compare.  It does not matter which you convert as long as they are the same.

 

The train traveling from Chicago is going 14.6 km/hour faster.

 

  1. The maximum pressure that can be sustained by a piston is 17.3 atm.  There is currently a pressure of 15.62 bar.  How many mmHg can be added before the pressure reaches the maximum?

Solution: To solve this problem, you must convert all of the pressures to atm.  From there, only subtraction is needed to solve the problem.

1428.8 mmHg can be added to reach the maximum pressure.

 

  1. Fill in the missing values:

0.834 N =? kg-m/s2 =? g-cm/s2 =? lbf

Solution: Use the conversions for force to fill in the values.

 1 N = 1 kg-m/s2 = 105 g-cm/s2 = 0.22481 lbf

0.834 N = 0.834 kg-m/s2 = 83400 g-cm/s2 = 0.1875 lbf

Simply multiply the value in Newtons by the conversion factors to get the new values.
